## Function/Symbol: draw.TextShadow

> Draw text with a shadow at x, y

### Required Context

- Parameters: x, y, text
- Requires: draw.Color, draw.SetFont (optional)

### Curated Usage Examples

```lua
draw.Color(255, 255, 255, 255)
draw.TextShadow(10, 10, "Shadowed text")
```

### Notes

- Good for readability on bright backgrounds
- Set font beforehand if custom font needed
