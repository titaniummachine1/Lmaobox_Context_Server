## Function/Symbol: engine.Notification

> Create TF2 client notification popup

### Required Context

- Parameters: title, longtext (optional)

### Curated Usage Examples

#### Simple notification

```lua
engine.Notification("Aimbot", "Target acquired")
```

#### Large text notification

```lua
engine.Notification("Script Info", "Full script details here...")
```

### Notes

- Shows in TF2's notification system
- Use sparingly to avoid spam
